# 2. Руководство пользователя

---

## 2.1 Как начать работу

Для начала работы с приложением:

1. Откройте веб-приложение в браузере  
2. Дождитесь загрузки интерфейса  
3. Перейдите на страницу регистрации или входа  

> В текущей версии MVP реализована регистрация и авторизация пользователей.  
> Для работы с системой необходимо создать аккаунт и войти в него.

---

## 2.2 Как ориентироваться в интерфейсе

Интерфейс приложения состоит из следующих основных элементов:

- Главная страница — отображает список всех задач  
- Форма создания/редактирования — используется для добавления и изменения задач  
- Личный кабинет — содержит статистику пользователя  
- Страница пользователей — список всех пользователей системы  
- Страница друзей — отображает добавленных друзей  

Элементы управления (кнопки):
- создание задачи  
- редактирование  
- удаление  
- добавление в друзья  

Навигация осуществляется через переходы между страницами.

---

## 2.3 Основные действия пользователя

### Регистрация
Для регистрации необходимо:

1. Перейти на страницу регистрации  
2. Заполнить данные (имя, email, пароль и др.)  
3. Подтвердить регистрацию  
4. Перейти на страницу входа  

---

### Вход в систему
Для входа:

1. Ввести email и пароль  
2. Нажать кнопку входа  
3. После успешной авторизации пользователь попадает на главную страницу  

---

### Создание задачи
Для создания задачи необходимо:

1. Перейти к форме добавления задачи  
2. Заполнить данные (название, объем работы и др.)  
3. Нажать кнопку создания  
4. Задача появится в списке  

---

### Редактирование задачи
Для редактирования:

1. Найти задачу в списке  
2. Нажать кнопку редактирования  
3. Изменить данные  
4. Сохранить изменения  

---

### Удаление задачи
Для удаления задачи:

1. Найти задачу  
2. Нажать кнопку удаления  
3. Подтвердить действие  

Задача будет удалена из системы.

---

### Работа с друзьями

#### Добавление друга
1. Перейти на страницу пользователей  
2. Выбрать пользователя  
3. Нажать кнопку добавления  

#### Удаление друга
1. Перейти на страницу друзей  
2. Выбрать пользователя  
3. Нажать кнопку удаления  

---

## 2.4 Как организовывать данные

В текущей версии MVP доступны базовые способы организации:

- Использование статуса задачи:
  - выполнена  
  - в процессе  

- Разделение задач по:
  - пользователю (каждый видит свои задачи)  
  - содержанию (название и описание)  

> Категории, теги и проекты в MVP не реализованы.

---

## 2.5 Как искать и фильтровать

В MVP предусмотрены базовые возможности:

- Просмотр всех задач  
- Разделение по статусу (выполненные / активные)  
- Просмотр задач конкретного пользователя  

> Полноценный поиск по ключевым словам отсутствует.

---

## 2.6 Как работают уведомления

В текущей версии MVP:

- Уведомления отсутствуют  
- Система не отправляет напоминания  

> В будущем возможно добавление:
- push-уведомлений  
- email-уведомлений  
- напоминаний  

---

## Примечание

Данное руководство описывает функциональность уровня MVP и может быть расширено по мере развития приложения.
